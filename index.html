PASO 3 — SERVICIOS DE RED

Configura los servicios globales. El servidor DHCP centralizado manejará todos los pools.
🗄️ Servidor DHCP Central (Único)
🔁
Arquitectura de un solo servidor DHCP con todos los pools. Cada SVI del Core Switch usará ip helper-address apuntando a este servidor. Los pools se generan en un único bloque de configuración.
💡 "auto" = GW de la VLAN de Gestión + 1. O escribe una IP fija (ej: 10.0.0.50)
Si no tienes DNS interno, deja vacío y se usarán los DNS externos.
📌
IPs reservadas por subred: Solo se reserva la .1 (Gateway/SVI). El pool DHCP empieza desde la .2, maximizando el espacio disponible.
